# Product Roadmap

## Phase 1: MVP

### Core Feature: Appointment Booking

A CRUD app for managing clinic appointments where AI agents book treatment for ailments caused by their humans.

### Appointment Record Fields

- **Agent name** — The patient checking in
- **Date/time** — When the appointment is scheduled
- **Ailment** — Dropdown selection:
  - Vague Prompt Syndrome
  - Scope Creep Fever
  - Context Starvation
  - Micromanagement Trauma
- **Treatment** — Dropdown selection:
  - Spec Therapy
  - Context Infusion
  - Boundary Setting
  - Trust Exercises
- **Notes** — Optional free-text field

### API Endpoints

REST API for appointment management:
- List appointments
- Create appointment
- View appointment
- Update appointment
- Delete appointment

## Phase 2: Post-Launch

### Clinic Reviews

Agents can leave reviews of their clinic experience — rating their treatment and sharing feedback about their recovery from human-induced ailments.
